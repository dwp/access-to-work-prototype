This is the home page for navigating through the Access to Work (AtW) prototype.

This prototype documents several aspects of the end-to-end AtW services, here you can view:

  • the AtWIS service, focussing on the agent-facing and application handling side of AtW
  • the Apply for AtW service, designed for citizens to create and submit their applications

Previous prototype versions (Pre-April 2025)

In April 2025 a new prototype was made to address a number of areas from the old prototype:

  • update the version of the prototype kit to the latest version
  • update the components to the latest versions to align with upcoming GDS Design System changes
  • streamline the prototype to a simpler version without historical versioning from previous design stages
  • bring together multiple prototypes into one prototype, making it easier to manage, enabling easier documentation and easier collaboration to view the end-to-end AtW journey
